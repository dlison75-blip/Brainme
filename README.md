## Getting Started

1. Simply open `index.html` in your web browser
2. Start typing your thoughts in the input box
3. Click "Save Thought" or press `Ctrl+Enter` (or `Cmd+Enter` on Mac) to save
4. Your thoughts are automatically saved in your browser's local storage

## Usage Tips

- **Quick Save**: Press `Ctrl+Enter` (Windows/Linux) or `Cmd+Enter` (Mac) to save quickly
- **Search**: Use the search box to filter through your thoughts
- **Sort**: Change the sort order to view newest or oldest thoughts first
- **Delete**: Click the delete button on any thought to remove it
